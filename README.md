# moviestore-backend-api

Movie rental API backend with user authentication, inventory management, and analytics. Built with TypeScript, Express.js, and JWT for my backend development learning journey.

## Features
- Secure JWT authentication with refresh tokens
- Role-based access (Customers & Staff)
- Complete movie inventory management
- Analytics dashboard for staff
- Overdue rental tracking
- Input validation & rate limiting

## Tech Stack
- **Runtime:** Node.js
- **Language:** TypeScript
- **Framework:** Express.js
- **Authentication:** JWT + bcrypt
- **Validation:** express-validator
- **Security:** Helmet, rate-limiting, httpOnly cookies

## Learning Outcomes
Through this project, I tried to get better understanding of:
- RESTful API design principles
- JWT-based authentication flows
- TypeScript for type-safe backend development
- Middleware patterns in Express.js
